# 🧠 Posture Detection System

A real-time posture monitoring system using Computer Vision and Full-Stack technologies. This project detects user posture through a webcam, identifies bad posture, and displays live analytics on a dashboard.

---

## 🚀 Features

* 🎥 Real-time posture detection using webcam
* 🧍‍♂️ Good vs Bad posture classification
* ⏱ Tracks current and total bad posture duration
* 🔊 Alert system for bad posture
* 🌐 Flask backend API
* 📊 React dashboard for live data visualization

---

## 🛠️ Tech Stack

### Backend

* Python
* Flask
* OpenCV
* MediaPipe

* React.js

## ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/Gowthami97568/posture-detection-system.git
cd posture-detection-system
```

---

### 2️⃣ Setup Backend

```
cd backend
pip install -r ../requirements.txt
python app.py
```

---

### 3️⃣ Run Frontend

```
cd frontend/frontend
npm install
npm start
```

---

### 4️⃣ Run AI Detection

```
python posture_detection.py
```
